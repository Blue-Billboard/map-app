**Part A: Replace the `<Steps>` component in the header template**

Find:
```html
<Steps v-model:activeStep="active" :model="items" :readonly="false" class="cursor-pointer" />
```

Replace with this custom indicator markup:
```html
<div class="wizard-step-indicator">
  <div
    v-for="(item, index) in items"
    :key="index"
    class="wizard-step-indicator__item"
  >
    <button
      class="wizard-step-indicator__block"
      :class="{
        'wizard-step-indicator__block--active': active === index,
        'wizard-step-indicator__block--complete': active > index,
        'wizard-step-indicator__block--pending': active < index
      }"
      @click="active = index"
    >
      <i v-if="active > index" class="pi pi-check" />
      <span v-else>{{ index + 1 }}</span>
    </button>
    <span class="wizard-step-indicator__label">{{ item.label }}</span>
  </div>
</div>
```

`Steps` is registered globally in main.ts — no import statement to remove from the script block.

**Part B: Wrap the three step containers in Vue Transition**

Find the `.wizard-content` div containing the three `v-if` step containers. Wrap ALL THREE with individual `<Transition>` tags using `:name="transitionName"`. Do NOT use `mode="out-in"` — both enter and leave must animate simultaneously:

```html
<div class="wizard-content">

      <!-- existing step 0 content unchanged -->
    </div>
  </Transition>
  <Transition :name="transitionName">
    <div v-if="active === 1" key="step-1" class="step-container">
      <!-- existing step 1 content unchanged -->
    </div>
  </Transition>
  <Transition :name="transitionName">
    <div v-if="active === 2" key="step-2" class="step-container">
      <!-- existing step 2 content unchanged -->
    </div>
  </Transition>
</div>
```

Add `key` attributes matching the step index on each step-container div.
