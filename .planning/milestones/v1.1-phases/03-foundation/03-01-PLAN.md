  <files>src/dialogs/wizard/WizardView.vue</files>
  <action>
Make two categories of CSS changes in the `<style scoped>` block of WizardView.vue.

**A. Apply Outfit font to wizard**

Add a new rule at the top of the scoped style block (after the opening `<style scoped>` tag):

```css
/* Flat Design System: Outfit font throughout */
:deep(.p-dialog.wizard-dialog),
:deep(.p-dialog.wizard-dialog) *,
.wizard-content,
.wizard-content * {
  font-family: 'Outfit', sans-serif;
}
```

**B. Remove dialog box-shadow**

Add a new deep override for the PrimeVue dialog container (PrimeVue injects this shadow — it is not in the scoped styles). Add after the Outfit font block:

```css
/* Flat Design System: no shadow on dialog */
:deep(.p-dialog) {
  box-shadow: none !important;
  border: 1px solid #e5e7eb;
}
```
